# Regression
Regression is a statistical method used in finance, investing, and other disciplines that attempts to determine the strength and character of the relationship between one dependent variable (usually denoted by Y) and a series of other variables (known as independent variables).

**Simple Linear Regression**

Simple Linear regression is the approximation of a linear model used to describe the relationship between two or more variables.
Dataset used is FuelConsumption of customer

**Multi Linear Regression**

Multiple linear regression (MLR) is to model the linear relationship between the independent variables and dependent variable.
Dataset used is FuelConsumption of customer

**Non Linear Regression**

Nonlinear regression is a form of regression analysis in which observational data are modeled by a function which is a nonlinear combination of the model parameters and depends on one or more independent variables.
Dataset used is Telephone calls of customer

**Polynomial Regression**

Polynomial regression is a form of regression analysis in which the relationship between the independent variable x and the dependent variable y is modelled as an nth degree polynomial.
Dataset used is FuelConsumption of customer
